Visual & Style Direction

Design Goals

  • Sleek, modern, and elegant
  • Reflective of NYC luxury and Zeshaan's professional personality
  • Minimalist layout with strong visual impact (bold imagery, clean fonts)

Color Usage Guidelines

  • Navy Blue should be the dominant brand color, used for key elements like navigation, headers, and important section backgrounds.
  • Gold should be used sparingly as an accent to highlight luxury aspects and create visual interest points.
  • Blood Orange creates dynamic contrast and should be reserved primarily for action items and interactive elements.
  • Teal can be used for secondary sections or supporting elements that require distinction without competing with primary colors.
  • Maintain appropriate color contrast ratios for accessibility, particularly with text elements.

Typography Guidelines

  • Maintain hierarchy clarity by using proper font sizes and weights consistently
  • Body text should be set at 16px minimum for optimal readability
  • Line height should be approximately 1.5-1.6 times the font size for body text
  • Limit use of italic styles except for specific emphasis needs
  • Ensure sufficient contrast between text and background colors
  • Use lighter font weights for larger display text, and medium to semibold weights for smaller text

Tone of Voice

Brand Voice Attributes

  • Professional, but warm and personal - Conveys expertise while remaining approachable
  • Knowledgeable and approachable - Demonstrates market insight without overwhelming jargon
  • Discreet, efficient, and client-first - Emphasizes service quality and respect for client privacy

Communication Guidelines

Do

  • Use concise, direct language that respects the reader's time
  • Balance professionalism with warmth and accessibility
  • Emphasize client benefits rather than features
  • Be specific about NYC neighborhoods and market knowledge
  • Maintain a consistent voice across all touchpoints

Don't

  • Use excessive jargon or technical terms without explanation
  • Adopt an overly casual or informal tone
  • Make exaggerated claims or use hyperbole
  • Use generic messaging that could apply to any real estate brand
  • Sacrifice clarity for cleverness

Form Styling Guidelines

Form Element Styling

  • Input fields: Clean borders (1px), subtle border-radius (4px), generous padding for touch targets
  • Labels: Clear, concise text positioned above inputs, semibold weight for visibility
  • Focus states: Subtle border color change to Navy Blue with light box shadow
  • Error states: Light red background, red border, with inline error message
  • Success confirmation: Subtle animation and color change to indicate successful submission

Form Behavior Best Practices

  • Implement inline validation to provide immediate feedback
  • Use proper input types for mobile optimization (email, tel, etc.)
  • Keep form length minimal while capturing essential information
  • Clearly mark required vs. optional fields
  • Provide helpful placeholder text that doesn't replace labels
  • Ensure form is fully accessible with proper ARIA attributes
  • Implement responsive design to maintain usability across all devices

Form Submission

After submission, users should receive:

  • Immediate visual confirmation of successful submission
  • A thank you message with expectations for follow-up
  • Automatic email confirmation sent to their provided email
